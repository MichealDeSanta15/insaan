Welcome all the OGs! I see you are back here with me for some more info about Mac or should I say Hackintosh!

Lets see some important requirements for installing Hackintosh:-

  • A laptop or a pc (quite obviously)
  • Intel core i3/i5/i7/i9 or AMD 3/5/7/9 (Threadrippers might work as well)
  • 8 GB of RAM (more is better)
  • SSD required, HDD might work
  • Most importantly, OPEN CORE

Let's come to the steps now:-

  1. Grab a laptop
  2. Install Open core
    3. The latest one is preferred
  3. Make an EFI file
    4. Make sure you make the correct SSDT for your system
  4. Now load it onto a USB stick
  5. Restart our pc with that stick
  6. Choose your efi to load the mac installer
  7. Install it
  8. VERY WELL, NOW DIAGNOSE ERRORS
  9. Good, you the OG HACKINTOSH installed
